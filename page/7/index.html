解密chrome浏览器账号密码

post @ 2024-11-29

通过文件来获取账号密码

从目标机器上获取需要的文件

将目标机器的masterkeys所在的文件夹拷贝到本地

将目标机器的chrome/edge浏览器的数据拷贝到本地（都是sqllite数据库格式的）

需要解密的就是Local State和Login Data以及Cookies（可能会被占用，可以尝试打快照或者其他拷贝方法）

从NTDS中获取dpapi backup keys

需要有ntds.dit文件和域控的system文件，需要用到DSInternals工具

获取dpapi backup keys（也可以使用对应用户的hash或者密码进行解密，不一定要使用dpapi backup keys）

使用SharpDPAPI解密masterkeys

使用SharpChrome获取AES State key

使用SharpChrome解密Login Data

通过dump目标内存来获取cookie

寻找目标chrome/edge主进程的pid

通过各种方式dump目标内存

使用CookieKatzMinidump.exe获取cookie

使用脚本将cookie注入到chrome中

通过chrome远程调试利用（not opsec）

kill现有的chrome，然后重新启动chrome并打开远程调试和恢复上一个session

注意：需要在对方的session下执行，也可以尝试修改目标快捷方式

远程调试的端口只开放在127.0.0.1，需要使用netsh将其转发到0.0.0.0

访问目标ip:9229/json可以看到对方访问的页面，也可以通过websocket接口dump cookie
